How to run ?

First run the Server socket on Terminal.  command : python socket_server.py --> Server will connect to the localhost.

Next command : python socket_client.py --> GUI will ask for the USERNAME, After entering username client will be connected with the same name.

Entry : Destination Cient = Username for communication
       
Entry : Message : Message to be sent 

		Here the local time will be appended with the Message string by "|". At the remote client it will be handled and compared to remote client's local time and the necessary action will be taken. And this will be reflected to terminal.
